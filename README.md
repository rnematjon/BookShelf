BookShelf is an online book store system created as a Final Project for Software Engineering Course

## Technologies Used

### Back-end
- Spring Boot
- Spring Security
- JPA
- MySQL
- Maven

### Front-end
- Thymeleaf
- Ajax
- jQuery
- Bootstrap

## Requirements

- `Java 11`
- `MySQL 5.7`

## Run

- Clone the project
- Create new MySQL database with name `bookdb`
- Run `BookShelfApplication`
- Test in localhost:8080
